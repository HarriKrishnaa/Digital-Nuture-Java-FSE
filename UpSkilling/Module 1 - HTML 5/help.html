Portal Help

This help page is adapted to match the portal's look and feel. Use the header links to browse main sections quickly.

  • Use the navigation bar to move directly to Home, Events, Gallery, Register, Feedback, Video, Location, and Contact.
  • Double-click gallery images to enlarge them, then click again to restore.
  • The registration form stores your preferred event type and warns about unsaved form changes.
  • Click the geolocation button in Location to find the nearest event from your browser.
  • Feedback includes a live character count and category-based guidance.

How to use this portal

  • Home: overview and quick tips.
  • Events: browse current community events.
  • Gallery: view event photos with enlarge support.
  • Register: sign up and see the event fee immediately.
  • Feedback: submit event feedback or support requests.
  • Video: watch the portal promo.
  • Location: discover nearby events with geolocation.
  • Contact: get help desk and help document access.
