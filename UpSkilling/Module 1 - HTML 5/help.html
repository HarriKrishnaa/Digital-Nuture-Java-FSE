HTML Commands

HTML

Section Block

Use a semantic section to group related website content.

<section id="events">
  <h2>Community Events</h2>
</section>
HTML

Navigation Link

Connect users to another section of the same page.

<a href="#contact">Contact</a>
HTML

Image Element

Display gallery images with accessible alternate text.

<img src="image.jpg" alt="Community event">
HTML

Form Input

Collect user details in registration and feedback forms.

<input type="email" placeholder="name@example.com" required>
HTML

Video Player

Embed a local promo or event video inside the page.

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>
HTML

Table Layout

Present event schedules in rows and columns.

<table>
  <tr><th>Event</th><td>Market Walk</td></tr>
</table>

CSS Commands

CSS

Background Color

Apply the light beige site background.

background: #f7f3eb;
CSS

Flex Layout

Arrange items in a row or column with flexible spacing.

display: flex;
gap: 12px;
CSS

Grid Columns

Build a clean gallery layout with equal columns.

display: grid;
grid-template-columns: repeat(2, 1fr);
CSS

Padding

Create breathing room inside cards and sections.

padding: 32px;
CSS

Rounded Corners

Soften cards, panels, and buttons for the current design.

border-radius: 14px;
CSS

Media Query

Adjust the layout for smaller mobile screens.

@media (max-width: 600px) {
  .gallery-grid {
    grid-template-columns: 1fr;
  }
}

JavaScript Commands

JavaScript

Select an Element

Grab an element from the page before updating it.

document.getElementById("feedbackText");
JavaScript

Attach Event Listener

Run JavaScript when the user clicks or types.

button.addEventListener("click", handleClick);
JavaScript

Update Text

Change visible content dynamically after an action.

status.textContent = "Form submitted.";
JavaScript

Save to Local Storage

Store preferences so they remain after refresh.

localStorage.setItem("preferredEventType", value);
JavaScript

Function Declaration

Define reusable logic for validation or other page actions.

function validatePhone() {
  // validation logic
}
JavaScript

Page Ready Event

Initialize the interface after the HTML finishes loading.

document.addEventListener("DOMContentLoaded", initializePage);
